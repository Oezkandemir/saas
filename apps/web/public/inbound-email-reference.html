Required Settings:

  • Go to: https://resend.com/domains ‚Üí cenety.com ‚Üí Inbound tab
  • Match Pattern: support@cenety.com (or *@cenety.com)
  • Destination: https://cenety.com/api/webhooks/resend/inbound
  • Status: ‚úì Enabled (toggle must be ON)

2DNS (MX Records)

Test MX Records:

nslookup -type=MX cenety.com

Expected Output:

cenety.com mail exchanger = 10 feedback-smtp.resend.com
If not showing:
  1. Go to your DNS provider (Cloudflare, Namecheap, etc.)
  2. Add MX record: Type=MX, Name=@, Value=feedback-smtp.resend.com, Priority=10
  3. Wait 15-30 minutes for DNS propagation
  4. Test again

3Vercel Environment Variables

Required for PRODUCTION:

  • RESEND_API_KEY = re_xxxxxxxxxxxxx
  • NEXT_PUBLIC_APP_URL = https://cenety.com
  • SUPABASE_SERVICE_ROLE_KEY = xxxxx...
  • NEXT_PUBLIC_SUPABASE_URL = https://xxx.supabase.co
  • NEXT_PUBLIC_SUPABASE_ANON_KEY = xxxxx...
After adding/changing variables:
  1. Go to Deployments tab
  2. Click latest deployment ‚Üí "..." ‚Üí Redeploy
  3. Uncheck "Use existing build cache"
  4. Wait 2-3 minutes
